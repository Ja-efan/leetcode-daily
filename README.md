# LeetCode Daily

1일 1리트코드 챌린지 기록 저장소  
목표: 하루 한 문제. 꾸준함. 기본기. 깔끔한 코드

## Rules
- 하루 최소 1문제 (Easy/Medium 위주, 가끔 Hard)
- 풀이 후 반드시 **복기(Takeaways)** 작성
- 틀렸던 문제/시간 오래 걸린 문제는 **Revisit**로 재풀이
- 커밋 메시지 규칙: `<state>: LC-<id> <title> [Easy|Medium|Hard]`

예) `solved: LC-1 Two Sum [Easy]`

## Structure
```
.
├── solutions/
│   ├── python/
│   │   ├── LC0001.py
│   │   └── …
│   └── javascript/
├── notes/
│   ├── patterns.md
│   └── revisit.md
└── log.md
```

## Log
- 진행 기록은 `log.md`에 누적한다.

## Notes
- `notes/patterns.md`: 자주 나오는 패턴(투포인터, 슬라이딩윈도우, DP 등) 정리
- `notes/revisit.md`: 다시 풀 문제 목록 + 재풀이 날짜 + 개선점

## Problem Template
각 솔루션 파일 상단에 아래 포맷으로 남긴다.

- Link:
- Approach:
- Time / Space:
- Edge cases:
- Takeaways:
- Revisit? (Y/N)
